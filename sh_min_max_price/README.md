About
============
This module useful to set minimum and maximum selling price for the product. The salesperson can easily see the minimum and maximum sale price so that will use to make clear action in sales procedure without waiting for a senior person.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list. 
5) search module name and hit install button.
