Minimum And Maximum Price Of Product

This module useful to set minimum and maximum selling price for the product. The salesperson can easily see the minimum and maximum sale price so that will use to make clear action in sales procedure without waiting for a senior person.

Key features:
  • Easy to set minimum and maximum price.
  • Validation on sale order save and confirm to check price set for that products.
  • Additional access rights groups added for procedure with less or greater than price specified.
  • This app is compatible with Odoo's Multi-Company Feature.

Version

v14.0.1 (16th October 2020)
  • Initial Release

For user Sales Manager "Allow confirm product sale price". This access rights made to confirm qutation till if its not validating maximum or minimum sale price of product.

Set Minimum Sale Price and Maximum Sale Price in Product.

In Sale Order or Quotation at Product Order Line on product price change, If its not validating maximum and minimum sale price rule than it shows warning message. It will just show message. User can be able to save this quotation.

Products which price unit breaking minimum or maximum price rule will be in red text.

On "confirm sale" if User have not "Allow confirm product sale price" access rights, It will not allow to confirm that quotation. It will raise error. Only "Allow confirm product sale price" access rights user can be able confirm it.
